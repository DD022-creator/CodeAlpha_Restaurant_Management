# 🍽️ Restaurant Management System - Task 3

## 📌 Project Overview
A Restaurant Management System built with **Node.js**, **Express.js**, and **MongoDB** as part of the **CodeAlpha Backend Development Internship**.

## ✨ Features
- ✅ Add/View menu items
- ✅ Add/View tables
- ✅ Place orders
- ✅ Track order status
- ✅ Auto-update inventory
- ✅ Table availability management
- ✅ REST APIs for everything

## 🛠️ Technologies Used
- **Node.js** - Backend runtime
- **Express.js** - Web framework
- **MongoDB** - Database
- **Mongoose** - ODM for MongoDB
